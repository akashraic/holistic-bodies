For Developer Use
This app was made using React Js, Node js, Npm Packages, and firebase. The components folder contains all the components and the assests folder contains the pictures and videos
You will need to have introductory familiarity with React js in order to understand how the structure works

A Node js terminal will be needed to create the app and download the NPM packages

To learn how to create and import react apps, please refer to Readme.md or find it in the react js documentation

to host on firebase, the website must be converted to static, this is done using the "react run build". to deploy on firebase, you must have proper access and setup your initialization on the terminal. 
https://github.com/woniesong92/firebase-hosting-guide 
all Npm Packages have comments in the code with their respective links.
